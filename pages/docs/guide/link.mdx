# Next.js 링크

모든 상대적 Markdown 링크는 자동으로 Next.js 링크로 변환됩니다. 이는 대상 페이지가 미리 가져와질 것임을 의미합니다. 그리고 링크를 클릭하면 전체 페이지 로드 없이 SPA처럼 클라이언트 측에서 페이지가 로드됩니다. 예를 들어:

```md
자세한 내용은 [여기](/about)를 클릭하세요.
```

이는 다음과 동일합니다:

```jsx
import Link from 'next/link'

자세한 내용은 <Link href="/about">여기</Link>를 클릭하세요.
```

이 기능으로 Nextra 페이지 간 이동이 빠르고 원활해집니다.
